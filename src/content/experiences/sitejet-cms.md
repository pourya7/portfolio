## Sitejet CRM and Website Builder

As a **Senior Frontend Engineer** at Sitejet (Berlin, Remote), I led the frontend engineering squad to overhaul the core website-builder and CRM user experience. My architecture work contributed to Sitejet's successful **acquisition by Plesk**.

### Key Achievements

- Led the **architectural migration** to modern React hooks and adopted a scalable design system
- Reduced **page-load time by 40%** and halved the development cycle time for new components
- Led the frontend engineering squad to overhaul the core **website-builder and CRM** user experience
- Contributed key improvements that helped drive the company's **acquisition by Plesk**
